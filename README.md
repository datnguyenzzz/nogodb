### NogoDB

>_**Disclaimer**: This project is under active development and deliberately crafted by human and chisel rather than AI. I have no desire to let AI steal the enjoyment of building things by hand. Although AI can produce code faster and boost "__productivity__", that is beside the point. This project is meant to entertain me and deepen my understanding of the database world, not to be rushed into a commercial product and making money from it :)_
